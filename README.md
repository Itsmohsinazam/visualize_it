# visualize_it

A modern web application for data visualization built with React, TypeScript, and Vite. This project demonstrates best practices for building interactive visualization tools with a focus on performance and user experience.

## 🎯 Overview

**visualize_it** is a TypeScript-based visualization project that combines React's powerful component system with Tailwind CSS for styling and Lucide React for beautiful icons. It provides a foundation for building dynamic, interactive data visualization dashboards.

## 🛠️ Tech Stack

- **Frontend Framework**: React 18.3.1
- **Language**: TypeScript 5.5.3
- **Build Tool**: Vite 5.4.2
- **Styling**: Tailwind CSS 3.4.1
- **Icons**: Lucide React 0.344.0
- **Database**: Supabase (Real-time database support)
- **Code Quality**: ESLint with TypeScript support
- **Development**: PostCSS for CSS processing

## 📊 Language Composition

- TypeScript: 87.5% - Core application logic
- JavaScript: 7.5% - Configuration and utilities
- HTML: 4.6% - Markup structure
- CSS: 0.4% - Styling

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Itsmohsinazam/visualize_it.git
cd visualize_it
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## 📝 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run typecheck` - TypeScript type checking
